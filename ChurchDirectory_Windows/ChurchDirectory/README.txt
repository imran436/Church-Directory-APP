THE GATHERING CHURCH — DIRECTORY GENERATOR
==========================================

WINDOWS
-------
1. Unzip the folder anywhere (Desktop works great)
2. Double-click  INSTALL_AND_RUN.bat
3. The app opens automatically

   First time only: if it says Python is not installed,
   click the link it opens, install Python (check
   "Add Python to PATH"), then double-click again.


MAC
---
1. Unzip the folder anywhere
2. Right-click  INSTALL_AND_RUN.command → Open → Open
   (Mac security prompt — you only do this once)
3. The app opens automatically

   First time only: if it says Python is not installed,
   click the link it opens, install Python, then
   double-click again.


EVERY TIME AFTER THAT
---------------------
Just double-click  INSTALL_AND_RUN  and the app opens.
No Python knowledge needed. No command line needed.


GENERATING THE DIRECTORY
------------------------
1. First launch asks for your Planning Center credentials
   (App ID and Secret — ask your admin)
2. Click  Generate Directory
3. The directory opens in your browser automatically


PRINTING
--------
When the directory opens in your browser:
  Ctrl+P  (Windows)  or  Cmd+P  (Mac)
  → Under Layout or Finishing, select  Booklet
  → Click Print
  → Fold the stack in half, staple the centre spine


PROBLEMS?
---------
Contact your administrator.
